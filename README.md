## 🚀 О проекте

    Программа написана командой из 5 человек на C# и скомпилирована в один исполняемый файл (.exe).
    Нам удалось спроектировать режим пользователя и администратора.

## 🔧 Системные требования

- **Операционная система:** Windows 10 / 11 (или Windows 7/8 с установленными обновлениями).
- **Платформа:** x64 или x86.
- **Зависимости:**
  - Требуется установленный [.NET 10.0 Runtime](https://dotnet.microsoft.com/ru-ru/download/dotnet/10.0) (или другая версия выше).

## 🎯 Как запустить

Запустить программу очень просто. Достаточно выполнить два шага:

1. **Скачай** архив `BOKIC3-1.0.0.zip` по ссылке выше.
2. **Разархивируй** архив `BOKIC3-1.0.0.zip`
3. **Перейди** в директорию `BOKIC3-1.0.0\BOKIC3\bin\Debug\net10.0`
4. **Запусти** `BOKIC3.exe`.
