## 🔧 Системные требования

- **Операционная система:** Windows 10 / 11 (или Windows 7/8 с установленными обновлениями).
- **Платформа:** x64 или x86.

## 🎯 Как запустить

Запустить программу очень просто. Достаточно выполнить два шага:

`Windows`

1. **Скачай** архив `BOKIC3.zip` по ссылке выше.
2. **Разархивируй** архив `BOKIC3.zip`
3. **Запусти** `BOKIC3.exe`.

`Linux (distributive)`

1. **Скачай** архив `BOKIC3-Linux.zip` по ссылке выше
2. **Разархивируй** `BOKIC3-Linux.zip`
3. **Открой** разархивированную директорию в терминале
4. **Сделай** файл исполняемым через команду `chmod +x BOKIC3`
5. **Запусти** `./BOKIC3`
