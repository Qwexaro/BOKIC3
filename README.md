## 🔧 Системные требования

- **Операционная система:** Windows 10 / 11 (или Windows 7/8 с установленными обновлениями).
- **Платформа:** x64 или x86.
- **Зависимости:**
  - Требуется установленный [.NET 10.0 Runtime](https://dotnet.microsoft.com/ru-ru/download/dotnet/10.0) (или другая версия выше).

## 🎯 Как запустить

Запустить программу очень просто. Достаточно выполнить два шага:

1. **Скачай** файл `BOKIC3.exe` по ссылке выше.
2. **Запусти** его двойным щелчком мыши.
